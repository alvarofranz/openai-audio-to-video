Describe the character types that appear in your story and what they should look like.

Each sequence will be preloaded with a prompt to generate the image. You can of course edit it, but it's a hard work to explain each prompt, so you need a solid starting base. That's what the image prompt preprocessing feature does. You can reference the following items here:

  • [image_style]: This contains your details for how the images should look like in general.
  • [character_types]: This contains the information for how you want to define your characters styles in general.
  • [story_items_style]: This contains an auto-generated list of items and their style, which will be generated by the OpenAI API, based on your current story.
  • [story]: This will contain the full story in text version, for context.
  • [current_sequence]: This will contain the narrated text that appears in the current scene, and is useful so the prompt processor knows what to focus on most.
